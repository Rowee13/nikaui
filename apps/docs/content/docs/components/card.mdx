---
title: Card
description: A card container with header, content, and footer sections.
---

## Installation

```bash
npx nika add card
```

## Usage

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
} from "@/components/ui/card"

export default function Example() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description text.</CardDescription>
      </CardHeader>
      <CardContent>
        <p>Card content goes here.</p>
      </CardContent>
      <CardFooter>
        <p>Card footer</p>
      </CardFooter>
    </Card>
  )
}
```

## Animation

Enable entrance animation with the `animated` prop:

```tsx
<Card animated>
  Animates in with a spring slide-up effect.
</Card>
```

## API Reference

### Card

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `animated` | `boolean` | `false` | Enable entrance animation |

### Sub-components

- `CardHeader` ‚Äî Container for title and description
- `CardTitle` ‚Äî Card heading
- `CardDescription` ‚Äî Card subtitle text
- `CardContent` ‚Äî Main content area
- `CardFooter` ‚Äî Bottom section

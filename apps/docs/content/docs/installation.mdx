## Prerequisites

Nika UI requires the following in your project:

- **React 18+** (19 recommended)
- **Tailwind CSS v4**
- **TypeScript** (recommended)

## Initialize Nika UI

Run the `init` command to set up Nika UI in your project:

```bash
npx nika init
```

This will:

1. Ask where to install components (default: `src/components/ui`)
2. Ask where to install utilities (default: `src/lib`)
3. Create `nika.config.ts` with your settings
4. Install base dependencies (`clsx`, `tailwind-merge`, `motion`)
5. Create the `cn()` utility and motion presets

## Add Components

Install components individually:

```bash
npx nika add button
npx nika add card dialog tabs
```

Each component is copied into your project with its dependencies automatically resolved.

## CLI Commands

| Command | Description |
| --- | --- |
| `npx nika init` | Initialize Nika UI in your project |
| `npx nika add <name>` | Add one or more components |
| `npx nika add <name> --overwrite` | Overwrite existing components |
| `npx nika list` | List all available components |

## Project Structure

After installation, your project will look like this:

```
src/
  components/
    ui/
      button.tsx
      card.tsx
      dialog.tsx
      ...
  lib/
    utils.ts      # cn() helper
    motion.ts     # Animation presets
nika.config.ts    # Configuration
```

## Configuration

The `nika.config.ts` file controls where components are installed:

```ts
export default {
  style: "default",
  tailwind: {
    css: "./src/app/globals.css",
  },
  aliases: {
    components: "@/components",
    ui: "@/components/ui",
    utils: "@/lib/utils",
    hooks: "@/hooks",
  },
  motion: true,
} as const;
```

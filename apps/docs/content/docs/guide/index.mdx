## What is Nika UI?

Nika UI is a component library for React that combines **Tailwind CSS**, **Motion** (formerly Framer Motion), and **Headless UI** to deliver beautiful, accessible, animated components.

Inspired by [shadcn/ui](https://ui.shadcn.com), Nika UI uses a **copy-paste model** — you install components individually via CLI, and the source code is copied directly into your project. You own the code and can customize freely.

## Key Features

- **Tailwind CSS v4** — No external CSS. Theme with CSS variables.
- **Motion Animations** — Every interactive component has smooth spring animations.
- **Headless UI** — Accessible primitives from the Tailwind Labs team.
- **TypeScript** — Full type safety with strict mode.
- **CLI** — Install components individually with `npx nika add`.
- **22+ Components** — From buttons to dialogs, tabs to toasts.

## Quick Start

```bash
# Initialize in your project
npx nika init

# Add components
npx nika add button card dialog

# Use them
import { Button } from "@/components/ui/button"
```

## Philosophy

Every Nika UI component follows these principles:

1. **Animation by default** — Components come with motion built in, but every animation can be disabled with `animated={false}`.
2. **Own the code** — Components are copied into your project, not installed as dependencies. Customize everything.
3. **Tailwind-native** — No CSS-in-JS. No external stylesheets. Just Tailwind classes and CSS variables.
4. **Accessible** — Interactive components use Headless UI for keyboard navigation, focus management, and screen reader support.
